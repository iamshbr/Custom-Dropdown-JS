Key Code Description
Initialization 

                  const dropdownOne = useCustomDropdown("#custom-dropdown-one", {
                    options,
                  });
container "" The selector for the container element where the dropdown is initialized. It is required; without it, the custom dropdown will not initialize. 

                  #custom-dropdown-one
                  .custom-dropdown-one
options [{}] An array of objects where each object's label represents the displayed value, and the value contains the actual value. It is mandatory for each object to include both the label and value keys. 

                  const options = [
                    { label: "Value 1", value: 1 },
                    { label: "Value 2", value: 2 }
                  ];
selectionAll false "A boolean flag indicating whether the 'Select All' functionality is enabled. By default, it is set to false." 

                  "selectionAll": true
multiple false "A boolean flag to allow or disallow multiple selections in the dropdown. By default, it is set to false." 

                  "multiple": true
search false "A boolean flag to enable or disable the search functionality within the dropdown. By default, it is set to false." 

                  "search": true
defaultValue [{}] "An array representing the default selected values in the dropdown when it is initialized. It is optional. If no value is selected by default, the first value will be selected automatically." 

                  "defaultValue": [ { label: "Value 2", value: 2 },]
formatResult (({ label, value, params }) => {}) "A function to customize how the result selection is formatted. It has three parameters: the label, the value, and params. Params represent the values you pass when providing the options array. For example, if you want a color to apply to all the options, you can pass a params key with a color property inside it to use it." 

                  formatResult: function ({ label, value, params }) {
                    return ('
                        <div class="siteItem">
                          <span class="site_details">
                            <span class="siteColorBox" style="background-color: ${params.siteColor}"></span>
                            <span class="item-site-name">${params.siteName}</span>
                          </span>
                        </div>')
                  }
formatOption (({ label, value, params }) => {}) "A function to customize how the option is formatted. It has three parameters: the label, the value, and params. Params represent the object you pass when providing the options array. For example, if you want a color to apply to all the options, you can pass a params key with a color property inside it to use it. Param also give the selected property which states that if the option is true or not." 

                  formatOption: function ({ label, value, params }) {
                    return ('
                        <div class="siteItem">
                          <span class="site_details">
                            <span class="siteColorBox" style="background-color: ${params.siteColor}"></span>
                            <span class="item-site-name">${params.siteName}</span>
                          </span>
                          <input type="checkbox" ${params.selected && "checked"} />
                        </div>')
                  }
formatSelectionAll (({ selectionAllBtnBool}) => {}) "A function that defines the display format for the 'Select All' option. It has a selectionAllBool property, which is set to true when all the options are selected." 

                  formatSelectionAll: function (selectionAllBtnBool) {
                    return ('
                    <a href="#">
                      <span class="site_details">
                        <span class="item-site-name">All</span>
                      </span>
                      <input type="checkbox" ${selectionAllBtnBool && "checked"} />
                    </a>')
                  }
triggerSelectAll (({ bool}) => {}) "A function that is called when the 'Select All' option is clicked. It has a bool property, which is set to true when all the options are selected." 

                  triggerSelectAll: function (bool) {
                    return ('
                      // Do Something
                    ')
                  }
change ((selectedOptions) => {}) "A function to define custom behavior when the dropdown value changes. It has a selectedOptions property, which is array of selected options." 

                  change: function (selectedOptions) {
                    return ('
                      // Do Something
                    ')
                  }
Get Value getValue(); "The getValue function retrieves the currently selected value(s)." 

                  customDropdown.getValue();
Set Value setValue([{label: 'Value 1' value:1}]); "The setValue function allows setting the value(s) programmatically. This function accepts an array of objects, where each object must include both the value and label keys." 

                  customDropdown.setValue([{label: 'Value 1' value:1}]);
